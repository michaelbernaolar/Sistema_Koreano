# Inventario en la nube

Sistema de inventario y ventas desarrollado con Streamlit y Python.

## Funcionalidades
- Gestión de productos
- Registro de ventas
- Control de stock
- Generación de comprobantes
- Reportes de ventas

## Tecnologías
- Python
- Streamlit
- Pandas

## Ejecución local
```bash
streamlit run main.py
